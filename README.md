1️⃣ What is the difference between var, let, and const?
Ans: var is function scoped and it can be redeclared and reassigned. However, let is block scoped and it also can be redeclared and reassigned. Even though const is block scoped, it can not redeclared or reassigned.
2️⃣ What is the spread operator (...)?
Ans: The spread operator is an operator which expands arrays and objects into individual elements to copy or merge or pass through an function argument.
3️⃣ What is the difference between map(), filter(), and forEach()?
Ans: map() works on each element and returns a new array. Though filter() returns a new array, it returns only the elements that can pass the condition. However, forEach() only loops over the elements without returning. 
4️⃣ What is an arrow function?
Ans: An arrow function is the modern way to write functions using => instead of the function keyword. In case of a single expression there is no need for curly braces or return keyword.
5️⃣ What are template literals?
Ans: Template literals are strings that begins and ends with backticks instead of quotation mark. This allows to put variables or expressions directly inside a string using ${} and write multiline strings.
